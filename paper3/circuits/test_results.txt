Note on nargo prove / nargo verify
------------------------------------
nargo 1.0.0-beta.19 removed the `nargo prove` and `nargo verify` subcommands.
Witness generation (nargo execute) is now separate from proof generation.
Full SNARK proof generation requires the `bb` (barretenberg) binary from
AztecProtocol/aztec-packages at a version compatible with ACIR 1.0.0-beta.19.
bb 0.76.x is not compatible with this ACIR version (format mismatch).

The equivalent workflow for full proof generation (once compatible bb is available):
  nargo compile            # compile to ACIR
  nargo execute            # generate witness  (VERIFIED above -- exit 0)
  bb prove -b target/accountable_redaction.json \
           -w target/accountable_redaction.gz \
           -o target/proof
  bb verify -k target/vk -p target/proof
